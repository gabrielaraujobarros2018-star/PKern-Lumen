# PKern-Lumen
This repo contains the Legacy and New versions of PKern Used in Lumen OS. PKern was Discontinued in Early February to Replace it with the Linux Kernel. PKern Will Arrive Again but as a Emergency Kernel in Case Linux Kernel breaks.

## Characteristics
  1. Part of Early experimental builds from January 11/12 to Early February
  2. Discontinued on Early February
  3. Will arrive Again in Lumen Final M2 or in Early M3 as a Emergency Kernel
  4. Named NeoKern in early versions but Renamed to PKern in Later experimental builds of Lumen
  5. Last appeared in Early February or Late January

## Integration
  1. Put PKern in /lumen-motonexus6/fw/boot/main/emergency_kern
  2. Integrate PKern as a Emergency Kernel
  3. Good Compiling & Good integration Matters
  4. HW Support too (Target: ARMv7a Only and Phone Moto Nexus 6 Hammerhead)

## Compiling
Pkern mostly uses C, so run:
`make` for Building
`make clean` for Decompiling
